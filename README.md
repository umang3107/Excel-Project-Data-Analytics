# Personal Finance Dashboard

## Project Overview
This dashboard analyzes personal spending patterns, savings trends, and payment behavior using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting


## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Pivot Tables
- Dashboard Design
- Data Visualization
- KPI Reporting
- Insight Generation


## Dataset
The dataset contains personal finance transactions including:
- Income
- Expenses
- Categories
- Payment Methods
- Daily Spending Records

  
## KPIs
- Total Expenses
- Total Savings
- Highest Spending Category
- Monthly Spending Trend
  

## Business Questions Solved
- Which category had the highest spending?
- Which payment method was used the most?
- What are the daily spending trends?
- Which days showed peak expenses?
- How much savings remained after expenses?

  
## Key Insights
- Food category contributed 32% of expenses
- UPI was the most used payment method
- Spending peaked during weekends
